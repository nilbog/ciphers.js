# Ciphers.js

### Description
**_Ciphers.js_** is a series of jQuery-based functions that perform some different ciphers

#### Default Validations
Input Type | Checks For | Purpose
--- | --- | --- |
caesar64 | Proper email format `(/.+@.\..+/gi)` | test
de_caesar64 | Does not contain `<` or `>` characters | test
